 ### VirtualDOM 虚拟DOM
 ```
1.什么是VirtualDOM ？
* VirtualDOM是由普通的JS对象描述的DOM对象。
* 模板引擎可以简化视图操作，只能知道当前状态，并不能跟踪状态；而VirtualDOM可以跟踪状态，记录当前状态和上一次状态，进行比对减少性能开销。
* MVVM框架解决了视图和状态同步的问题。

2.VirtualDOM的作用 ？
* 可以跟踪DOM状态，记录当前状态和上一次的状态，比对后在进行最终的DOM修改，减少DOM操作次数，减少性能开销，提升性能。
* 维护视图和状态的关系
* 复杂视图情况下可以提升渲染性能【如jQuery和VirtualDOM在增删改查时对DOM的渲染完全不同，jQuery渲染效果差，VirtualDOM渲染效果好】
* 跨平台[浏览器PC端、服务器端、小程序、原生应用等]
 ```

 ### 虚拟DOM库：SnabbDOM
```
1.SnabbDOM简介：
* Vue.js 2.x内部使用的虚拟DOM就是改造的SnabbDOM；
* Snabbdom是一个轻量级的DOM操作库，可以快速的构建DOM树，并且可以跨浏览器进行渲染；
* SnabbDOM源码使用TypeScript开发；
* SnabbDOM是最快的虚拟DOM之一；
* SnabbDOM通过模块可以扩展功能；

2.使用：
全局安装parcel-bundler
①安装parcel-bundler : [模块配置："dev": "parcel index.html --open" ;"build": "parcel build index.html"]
②配置scripts
③搭建目录结构
④npm i snabbdom
```
